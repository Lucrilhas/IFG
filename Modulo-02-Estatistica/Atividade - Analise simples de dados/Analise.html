Analise 02 - Verificar se a disposição do grupo influencia na decisão de fumar¶

df_machos = df.loc[df['Sexo'] == 'M']
macho_count = df_machos.value_counts("Toler", normalize=True)

df_femeas = df.loc[df['Sexo'] == 'F']
femea_count = df_femeas.value_counts("Toler", normalize=True)

indexes = ["Incomoda pouco", "Incomoda muito", "Indiferente"]
macho_tol = [macho_count['P'], macho_count['M'], macho_count['I']]
femea_tol = [femea_count['P'], femea_count['M'], femea_count['I']]

fig, axs = plt.subplots(1, 2, constrained_layout = True)
fig.suptitle("Tolerncia ao fumo por gnero", fontsize=14)
axs[0].set_title("Tolerncia dos Homens")
axs[0].pie(macho_tol, labels=indexes, autopct='%1.2f%%')

handles, labels = axs[0].get_legend_handles_labels()
fig.legend(handles, labels)

axs[1].set_title("Tolerncia das Mulheres")
axs[1].pie(femea_tol, labels=indexes, autopct='%1.2f%%')
plt.show()

A partir do grfico acima, podemos chegar concluso de que, em geral, os homens tendem a se importar menos com fumantes ao seu redor, enquanto as mulheres demonstram maior preocupao.

Dado que o ambiente influencia nas aes do indivduo, as mulheres tendem a no fumar, j que a tolerncia ao seu redor no o permitiria.

Por outro lado, os homens enfrentaro uma presso menor e tero uma chance maior de se tornarem fumantes.

Essa concluso est em consonncia com a Anlise 1 e pode revelar uma das implicaes do resultado obtido.

dfA = df[df['Turma'] == 'A']
dfB = df[df['Turma'] == 'B']

turmas = ("Turma A", "Turma B")
preferencias = {
    'Cinema Bom': (dfA['OpCine'].value_counts().get('B', 0), dfB['OpCine'].value_counts().get('B', 0)),
    'Cinema Muito Bom': (dfA['OpCine'].value_counts().get('M', 0), dfB['OpCine'].value_counts().get('M', 0)),
    'TV ruim': (dfA['OpTV'].value_counts().get('R', 0), dfB['OpTV'].value_counts().get('R', 0)),
    'TV media': (dfA['OpTV'].value_counts().get('M', 0), dfB['OpTV'].value_counts().get('M', 0)),
    'TV boa': (dfA['OpTV'].value_counts().get('B', 0), dfB['OpTV'].value_counts().get('B', 0)),
    'TV ns': (dfA['OpTV'].value_counts().get('N', 0), dfB['OpTV'].value_counts().get('N', 0)),
}
print(preferencias)

x = np.arange(len(turmas))  # the label locations
width = 0.1  # the width of the bars
multiplier = 0

fig, ax = plt.subplots(constrained_layout = True, figsize=(8,6))

for attribute, measurement in preferencias.items():
    offset = width * multiplier
    rects = ax.bar(x + offset, measurement, width, label=attribute)
    ax.bar_label(rects, padding=1)
    multiplier += 1

# Add some text for labels, title and custom x-axis tick labels, etc.
